# 📊 EDA on Retail Sales Data

## Oasis Infobyte Data Analytics Internship

### Task 1 – Exploratory Data Analysis (EDA) on Retail Sales Data

---

## 📌 Project Overview

This project is part of the **Oasis Infobyte Data Analytics Internship**.

The objective of this project is to perform **Exploratory Data Analysis (EDA)** on a retail sales dataset to discover meaningful patterns, customer behavior, sales trends, and business insights using Python.

The analysis includes data cleaning, statistical summaries, visualizations, and actionable business recommendations.

---

## 🎯 Objectives

- Load and inspect the retail sales dataset
- Perform data cleaning and preprocessing
- Analyze sales trends over time
- Understand customer demographics
- Identify top-performing products and categories
- Discover relationships between numerical features
- Generate business insights through visualizations

---
